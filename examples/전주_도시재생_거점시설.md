# 전주시 도시재생 거점시설 — 주소 → PNU·좌표 → SHP 변환 기록
> 공개 소식지 PDF에서 거점시설 주소를 추출해 VWorld로 PNU·좌표를 뽑고 SHP(포인트)로 만든 작업 기록입니다.
- 원본: `2025 전주시 도시재생 소식지 「함께 만든 10년, 함께 열어갈 미래」_거점시설.pdf` (6쪽, 텍스트 추출형)
- 대상: 거점시설 **35곳**
- 지오코딩 성공: **34/35** (VWorld 지번(PARCEL)→도로명(ROAD) 순 시도)
- 좌표계: 원좌표 EPSG:4326 → SHP는 **EPSG:5186(중부원점)**

## 처리 파이프라인
1. **PDF 텍스트 추출** — PyMuPDF(`fitz`)로 6쪽 텍스트 추출 (스캔 아님, 선택가능 텍스트).
2. **주소 정제** — `일원`·`번지`·`외 N필지`·`(예정)` 제거, 다중지번은 첫 지번 사용.
3. **지오코딩** — VWorld `getcoord` 2.0, `type=PARCEL` 먼저 실패 시 `ROAD` (지번/도로명 혼재 대응). PNU(`level4LC`)·경위도·정제주소 취득.
4. **엑셀 생성** — openpyxl.
5. **SHP 변환** — GeoJSON(4326) → `ogr2ogr -t_srs EPSG:5186 -lco ENCODING=UTF-8`.

## 확인이 필요한 데이터 이슈
- **G-Town(1번)**: 덕진동1가 `12571-1` 지번이 비정상적으로 큼 → 지오코딩 실패(신축 예정지/PDF 오타 의심).
- **수다가든 1~4호(20~23)**: PDF에 `완산구 팔복동`으로 표기되나 팔복동은 **덕진구** → 덕진구로 정정해 성공.
- **14·15번(행복플러스센터/창작레지던시플라자)**, **31·33·34·35번**: 잡지형 2단 편집이라 이름↔주소 매핑 재확인 권장.
- **도로명주소 8곳**: 좌표는 있으나 PNU 없음(PNU는 지번에서만). 필요 시 좌표→필지 역조회로 PNU 보완 가능.
- **SHP 한글 필드명**: DBF 10바이트 제한으로 `건설유형→건설유` 등 잘림(값은 정상). 필드명 온전하려면 `.gpkg` 사용.

## 재현 방법
```
# 1) PDF 텍스트 추출
pip install pymupdf openpyxl requests
# 2) 주소 정제 후 VWorld getcoord(PARCEL→ROAD)로 지오코딩 → xlsx/geojson 생성
# 3) GeoJSON → SHP (QGIS GDAL)
ogr2ogr -f "ESRI Shapefile" -s_srs EPSG:4326 -t_srs EPSG:5186 -lco ENCODING=UTF-8 out.shp in.geojson
```
